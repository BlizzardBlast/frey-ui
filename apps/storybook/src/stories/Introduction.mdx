# Frey UI

A strict, accessible design system library for internal applications.

## What this Storybook is for

- Browse live component examples and states
- Verify behavior across light and dark themes
- Use Controls and a11y checks while developing

## Quick start

```bash
pnpm add frey-ui
```

```tsx
import 'frey-ui/style.css';
import { ThemeProvider, Button } from 'frey-ui';

export function Example() {
  return (
    <ThemeProvider theme='light'>
      <Button>Save changes</Button>
    </ThemeProvider>
  );
}
```

## Components in this library

- Alert
- Button
- Checkbox
- Chip
- Skeleton
- Switch
- TextInput
- ThemeProvider
