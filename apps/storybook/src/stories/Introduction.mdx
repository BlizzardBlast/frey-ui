# Frey UI

A strict, accessible design system library for internal applications.

## Installation

```bash
pnpm add frey-ui
# OR
npm install frey-ui
```

## Setup

Import the CSS once in your root file:

```tsx
import 'frey-ui/style.css';
```

## Usage

```tsx
import { Chip, Switch, ThemeProvider } from 'frey-ui';

function App() {
  return (
    <ThemeProvider theme='light'>
      <Chip label='Status' variant='outlined' />
      <Switch label='Enable Dark Mode' />
    </ThemeProvider>
  );
}
```

## Nested Theming

`ThemeProvider` uses `data-frey-theme`, so nested light/dark regions are supported:

```tsx
<ThemeProvider theme='light'>
  <main>
    <ThemeProvider theme='dark'>
      <Switch label='Dark panel setting' />
    </ThemeProvider>
  </main>
</ThemeProvider>
```
